# 中国高校全景 · 2026

> 在一张地图上，读懂中国的好大学。

一个**纯静态、完全离线可用**的中国重点高校信息可视化项目：以中国地图呈现高校分布，支持按「层次 / 类型 / 特色院校群 / 地区」多维交叉筛选，逐校查看校徽、创立时间、所在地、优势学科（A+/A/A- 分级）、就业去向与「2025 重庆参考分数」，并内置「志愿填报 · 冲稳保」辅助查询。

数据整理截至 **2026 年 6 月 9 日**（高考最后一天）。

---

## ✨ 功能特性

- **全国分布地图**（ECharts，本地化离线）：
  - 「院校气泡」模式——按层次配色、按综合层级定大小，同城院校自动散开，悬停看简介、点击气泡看详情；
  - 「各省数量」模式——按当前筛选统计各省入选院校数的热力图；
  - 滚轮缩放、拖拽平移、「↩返回全国」一键复位；
  - **「随筛选联动」开关**——开启时筛选实时更新地图（仅轻量更新散点、不重建底图）；关闭后点击分类不再改动地图。
- **多维分类浏览 + 分面计数**：层次（清北 / 华五 / C9 / 985 / 211 / 双一流）、类型（综合 / 理工 / 师范 / 财经 / 政法 / 医药 / 农林 / 语言 / 民族 / 军事 / 海洋 / 艺术 / 体育）、特色院校群（国防七子 / 政法五院 / 两财一贸 / 两电一邮 / 新型研究型 / 部属师范 / 建筑老八校等）、所在地区（七大区）。同维「或」、跨维「且」，选中后其它维度数量实时更新。
- **志愿填报辅助 · 冲稳保**：选科类（物理 / 历史）+ 输入位次（推荐）或分数，按「冲 / 稳 / 保」三档匹配院校，结果可点开详情。
- **校徽与英文简称**：每所院校展示矢量校徽（无徽标时回退为中文字徽）；英文简称（如 BUPT、THU）以标签形式置于标签行首位。
- **搜索 / 排序**：按校名、城市、学科搜索；按综合层级、2025 物理 / 历史分数、创立时间、校名拼音排序。
- **单校详情抽屉**：校徽 + 基本信息、双一流建设学科、**教育部学科评估 A+/A/A- 三级分色**、2025 重庆物理 / 历史类参考分与位次、加深后的简介与就业概况。
- **首页国旗**：右侧矢量五星红旗装饰，窄屏自动退化为淡背景。
- **完全离线**：ECharts、中国地图 geoJSON、校徽均已本地化，双击 `index.html` 即可使用，无需联网、无需服务器。

- **校徽覆盖**：**188 / 188 全部有真徽标**（其中 170 所用新一批 `logos-new`，支持 SVG / PNG），无需字徽兜底。
- **英文简称**：188 / 188 均有显示。
- **学科评估**：约 140 / 188 有 A+/A/A- 学科；其余多为双非 / 艺术 / 医科 / 未参评院校，详情页如实标注「暂无 A 类学科」。

### ⚠ 关于分数（务必阅读）

- 页面所有分数均为 **2025 年重庆市普通类本科批「院校专业组」录取最低分的参考整理值**，物理类 / 历史类分列，**仅供大致定位**。
- **2026 年的真实录取分数线此刻并不存在**——将于 6 月下旬出分、7—8 月录取后才公布，届时可据官方数据更新。
- 军事 / 艺术 / 体育类院校多通过提前批、军检、校考或综合评价录取，普通类分数从略，显示「—」。
- 「志愿填报辅助」基于 2025 参考数据测算，**仅供定位，不构成填报建议**。一切以教育部、各高校及**重庆市教育考试院**公布的权威数据为准。

### 资料来源

院校创立时间、所在地、主管部门、学科实力（双一流学科、教育部第四轮学科评估 A+/A/A- 学科）等基于公开资料整理；标签分类依据教育部「双一流」建设名单与中国社会主流认知。校徽取自开源项目「[中国大学矢量校徽合集](https://github.com/)」（设计师 pluwen，矢量 SVG）。

## 🚀 使用方式

**方式一（最简单）**：直接双击 `index.html`，用浏览器打开即可（已完全离线）。

**方式二（本地服务器，推荐）**：在项目根目录执行任一命令，然后访问 `http://localhost:8765`。

```bash
python -m http.server 8765
#  或
npx serve .
```

> 修改代码后请用 `Ctrl + F5` 强制刷新，避免浏览器使用缓存的旧 JS / CSS。

## 🛠 如何维护数据

### 院校数据 schema

每所院校是一条 JS 对象，字段含义见 `data/universities.js` 顶部注释。核心字段：

```js
{
  id:'tsinghua', name:'清华大学', en:'Tsinghua University', abbr:'清华', enAbbr:'THU',
  province:'北京', city:'北京', lng:116.326, lat:40.003, founded:1911,
  type:'综合', admin:'教育部',
  tags:['清北','C9','985','211','双一流'],   // 层次 + 特色群标签
  tier:0,                                     // 0清北 1华五 2其他C9/人大/国防科大 3其他985 4强211/特色 5其他双一流
  firstClass:[...],   // 双一流建设学科
  aplus:[...], a:[...], aminus:[...],         // 教育部第四轮学科评估 A+/A/A- 学科（高→低，多则精选少则全列）
  score:{ wl:{min:695,rank:80}, ls:{min:660,rank:120} },  // 2025重庆 物理类/历史类，无则置 null
  emp:'就业概况（2句）…', intro:'简介（2-3句）…'
}
```

- **新增院校**：在 `data/` 下任一 `uni-*.js` 的 `push(...)` 列表追加；或新建文件并在 `index.html` 末尾、`tools/*.js` 加载列表中各加一行。
- **新增分类标签 / 类型**：在 `data/meta.js` 的 `levels` / `types` / `groups` 中登记（含中文名与配色），筛选 UI、图例与卡片会自动识别。

### 校徽

- 校徽分两套：`assets/logos/`（原始套图，保留不动）与 `assets/logos-new/`（新一批），同名规则均为 `中文校名 英文简写.svg`。**显示时优先用 `logos-new`，缺失则回退 `logos`，再缺失用中文字徽兜底。**
- 新增 / 替换校徽：把文件命名为 `中文校名-logo.svg`（或已知英文名）放到项目根目录，运行 `node tools/gen-logos.js`，脚本会自动改名归入 `assets/logos-new/`、清理根目录，并重新生成 `data/logos.js` + 打印覆盖率。校名用字不一致时，在脚本的 `ALIAS` 表里加映射即可。

### 更新分数（2026 出分后）

把对应院校的 `score.wl` / `score.ls` 改为 2026 真实数据，并把 `data/meta.js` 里 `meta.scoreYear`、`meta.scoreNote` 的年份改为 2026，页面所有「2025 参考」字样会随之更新。

### 校验

```bash
node tools/validate.js   # 字段完整性、id 唯一性、类型/省份/标签合法性 + 各维度统计
node tools/smoke.js      # 桩掉 DOM 跑通页面初始化与渲染，捕捉运行时错误
```
